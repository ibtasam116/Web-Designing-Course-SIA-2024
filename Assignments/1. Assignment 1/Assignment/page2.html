Attributes
The HTML attributes are used to provide more information to an HTML elements.

  • HTML attributes only appear at start tags. It will never be on end tags.

  • All HTML elements can have attributes.

  • HTML elements can have multiple attributes.

  • HTML attributes usually come in name/value pairs like: name="value"

    • An attribute name

    • An equal = sign

    • A value surri=ounded by quotation marks "value"

when the value contains double quotes.

There are two ways to specify the URL in the href attribute.

  1. Absolute URL: Link to an external website that is hosted on another server.

  2. Relative URL: Link to an html page that is hosted within the website. Here, the URL does not include the domain name. If the URL begins without a slash, it will be relative to the current page. Example:
    href="login.html".
