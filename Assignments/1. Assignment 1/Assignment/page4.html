In HTML5 there are some new semantic elements that can be used to
define different parts of a web page.

  •       <article>

  •       <aside>

  •       <details>

  •       <figcaption>

  •       <figure>

  •       <footer>

  •       <header>

  •       <main>

  •       <mark>

  •       <nav>

  •       <section>

  •       <summary>

  •       <time>
